Chat Plays — You Control the Game!

Type commands in Twitch chat and watch Pokémon LeafGreen move on stream. Everyone plays together!

ℹ️ Sometimes Chat Plays is in test mode (only a few testers can play) or takes a short break. If your commands don't do anything, that's all it is — nothing's broken, and it'll be back!

How to play

One command does it all: !p followed by the buttons you want to press.

Press a button

!p a

Presses the A button once. That's it — you just played!

Hold a button — add a number of seconds

!p up 2

Holds Up for 2 seconds, so your character walks up. You can hold any button for up to 3 seconds.

Chain a few together (up to 5)

!p up 2 a b

Walks up for 2 seconds, then presses A, then presses B. Each number sticks to the button right before it.

The buttons you can use

TypeWhat it does in the game
up down left rightMove around / pick menu options
aConfirm / talk / interact
bCancel / back (also run, with running shoes!)
startOpen the menu
selectUse a registered item
l rThe shoulder buttons

Anything else simply won't work — the bot only understands these.

Good to know

• Wait a few seconds between commands — everyone shares the controller, so there's a short cooldown.

• If lots of people are playing, commands line up in a queue and run in order.

• Some commands get skipped on purpose when they'd press a button at a bad moment — the bot is keeping the game safe. Just try again!

Points

You earn points automatically just by hanging out and chatting during the stream.

!points

Shows how many points you have. More ways to earn and spend them are coming soon — collect now, spend later!

The rules

Short and simple — this is a game we all share.

😊 Be kind and play fair

Chat Plays works because everyone takes turns and works together. Help each other figure out where to go next!

🛡️ The bot protects the game

Buttons that aren't on the list won't do anything, and the bot watches the screen and skips commands that would cause problems. You can't break anything by accident, so relax and have fun.

⚠️ Trying to cause trouble has consequences

Everything sent to the bot is logged, and the mods can see all of it. Deliberately trying to mess up the game, the save, or the stream will get commands ignored — and can mean a timeout or a ban. Don't be that person; it's never worth it.
